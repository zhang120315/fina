# 《摩根财团》英文学习笔记总索引

## 使用方式

- 阅读某一章时，把不会的单词、词组或原句发给我，并说明章节号，例如：`第 4 章：in the throes of / placate / state credit`。
- 我会把内容追加进对应章节笔记；每个表达单独成一个小节，按“原文语境 -> 核心意思 -> 用法拆解 -> 举一反三 -> 例句”整理。
- 如果你只发词而没说章节，我会先按最近一次正在处理的章节判断；不确定时再问你。
